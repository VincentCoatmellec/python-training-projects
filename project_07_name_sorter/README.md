# Name Sorter

This project is a simple Python script that reads a file containing names, sorts them alphabetically, and writes the sorted list back to a file.

## Description

This project is an introduction to working with data in Python.

We will still manipulate and organize data stored inside a file.

The program reads a file containing a list of names, sorts them in alphabetical order, and writes the sorted result back to the file.

This project shows how Python can be used to easily process and organize data.

## What this project practices

- Reading files in Python
- Writing files
- Working with lists
- Sorting data
- Basic data manipulation

## Requirements

- Python 3.13
- No external libraries

## How to run

```bash
python3 name_sorter.py
