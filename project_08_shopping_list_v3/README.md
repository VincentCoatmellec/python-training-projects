# Shopping List (CLI Version 3 - File Persistence)

This project is an improved version of the Shopping List application, refactored using object-oriented programming.

## Description

In this version, the shopping list is managed by a `Liste` class and saved to the local disk using a JSON file.

This project helps practice:

- Object-oriented programming (OOP)
- Reading and writing files in Python
- Working with JSON format
- Data persistence
- Managing program state
- User interaction and loops
