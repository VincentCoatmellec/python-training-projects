# Folder Structure Generator

This project is a small Python script that automatically creates a predefined folder structure.

## Description

In many situations, we need to create the same folder structure multiple times.  
For example:

- Organizing company accounting files
- Managing files for a project (like a movie or a 3D animation)
- Storing documents for employees

Creating these folders manually can be slow and repetitive.  
This script automates the process.

The folder structure is defined using a Python dictionary, and the program creates all the folders automatically.

## What this project practices

- Using dictionaries
- Working with folders in Python
- Using loops and basic logic
- Automating repetitive tasks

## Requirements

- Python 3.13
- No external libraries

## How to run

```bash
python3 folder_structure_generator.py
