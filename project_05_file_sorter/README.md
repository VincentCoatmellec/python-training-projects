# File Sorter

This project is a small Python script that sorts files based on their extension.

## Description

Sometimes folders like the Downloads folder quickly become messy with many different types of files.

This program automatically organizes files by moving them into folders according to their extension.

For example, images, documents, and videos can each be placed into their own folder.

This script is useful if you run it regularly to keep a folder organized.

The project also encourages the use of the **pathlib** library, which makes working with files and folders in Python easier than using the `os` module.

## What this project practices

- Working with files and folders in Python
- Using file extensions
- Using the `pathlib` library
- Loops and basic program logic
- Automating simple tasks

## Requirements

- Python 3.13
- No external libraries

## How to run

```bash
python3 file_sorter.py
